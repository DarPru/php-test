1 - В терминале переходим в директорию проекта и создаем новую ветку:
````
git checkout -b моя_новая_ветка
````

2 - Вносим изменения в файлы проекта и проверяем статус:
````
git status
````

3 - Добавляем изменения, затем коммитим:
````
git add .
git commit -m "Это мой коммит"

````
4 - Отправляем изменения в удаленный репозиторий:
````
git push origin мля_новая_ветка
````
